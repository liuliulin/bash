命令        说明
cut     通过分隔符拆分后,显示指定的区域
-b 以字节为单位进行分割
-c 以字符为单位进行分割
-d 自定义分隔符,默认为制表符
-f 与-d一起使用指定显示哪个区域
-n 取消分割多字节字符,仅和 -b 标志一起使用

grep    显示与模式相匹配的行
-i 不区分大小写
-n 显示匹配行和行号
-v只显示不匹配行
-c 只输出匹配行的行数
-h 查询多文件的时候不显示匹配的文件名字
-l 查询多文件只显示匹配的文件名
-s 不显示错误信息

grep与正则表达式同用
grep "^[^abc]" *.f 提取行首不是abc的行
grep "[0-9]\{3\}[8]" *.f 提取类似xxx8的行,x表示任意数字
grep "a\{2\}" *.f 显示a至少出现两次的行,注意grep是以行处理单位的
grep -n "^$" file 列出空行的行号
ps aux | grep "logi[n]" 

查找文件中指定的字符串
grep -Hr --binary-files=without-match "字符串"  路径
-H用于指定找到的文件名,-r选项指定递归所有目录, --banary-files=without-match排除二进制文件

head    显示文件的开始部分
tail    显示文件的结束部分
-f  持续监听文件

paste   通过指定的分隔符将两个文件的行进行合并,或者通过指定的分隔符合并一个文件中的多行
-d 指定分隔符
-s 并行显示

sort    对多行进行排序
-n    将数字识别为数字
-t    指定字段的分隔符(-t ':')
-k    指定的排序的字段
-r    降序
-g    按数值进行比较

tr      替换,删除字符,压缩文字序列

uniq    压缩连续的相同的行
-n    将数字识别为数字
-t    指定字段的分隔符(-t ':')
-k    指定的排序的字段
-r    降序
-g    按数值进行比较

wc      显示字节数,字数,行数
-c    字节数
-l    行数
-w    字数
